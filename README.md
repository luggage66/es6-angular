#weather-app

Trying to use es6 class and modules to write a basic angular app.

Tools:
*  gulp
*  node-babel
*  node
*  bower

Instructions:
-----------------

```
# install dependencies
npm install -g webpack
npm install

# build the client files
webpack

# start a server to host
node server.js
```

Open http://localhost:9000/index.html
